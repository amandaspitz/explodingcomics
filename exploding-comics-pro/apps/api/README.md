# Exploding Comics Pro API

Backend API for the professionalized Exploding Comics platform.

## Scope

This application will own:

- public comic read APIs
- administrative content APIs
- like persistence
- view persistence
- publication state
- share metadata

## Principles

- backend-first migration
- SDD with SOLID boundaries
- explicit validation
- centralized error handling
- testable modules

## Milestone 1 Deliverables

- TypeScript Express scaffold
- environment configuration
- health endpoints
- centralized error middleware
- baseline tests

## Commands

After dependencies are installed:

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run test`
- `npm run check`
- `npm run db:check`
- `npm run migrate`
